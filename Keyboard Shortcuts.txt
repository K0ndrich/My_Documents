Windows 

Win + L --- заблокировать пк

Ctrl + C --- скопировать текст

Ctrl + V --- вставить текст (screenshot)

Ctrl + S --- сохранить документ

Ctrl + Shift + B --- изменить текст на жирный в Word

Ctrl + A --- выделить весь текст в любом документе

Ctrl + R --- перезапускает сессию в google

Visual Studio Code

Ctrl + F --- найти слова в коде

Ctrl + H --- заменить слова в коде

Shift + Alt + F --- выравнивание кода

Shift + Alt + стрелочка верх / вниз --- копировани и втсавление строчки кода в верх / вниз

Ctrl + Shift + K --- удаление выбраной строки кода

Ctrl + Z --- возвращение удаленой ранее строки кода

Ctrl + K + C --- закоментировать выделеный участок кода

Ctrl + K + U --- разкоментировать выделеный участок кода

Ctrl + -  --- увеличивает ил уменьшает все значки програмы

Ctrl + Shift + P --- открывается глобальная командная строка

Ctrl + L --- сокпировать выбраную строку

Ctrl + X --- вырезание строки

Ctrl + F +++ Alt + Enter --- вибираем однаковие название и редактируме их

Alt + Shift --- выбирает сразу несколько строчек для редактирования

Ctrl + B --- свернуть боковую панель 

< } > --- вернуть посленднюю команду в терминале

Ctrl + ` --- открыть терминал в vs code (внутрений)
